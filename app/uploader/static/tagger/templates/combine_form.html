Select the text processing mode

Wordseer requires at least one XML element to be designated as a text node. The smallest unit of text analysis for this text is a sentence. Ideally, the selected text element must contain one or more sentences per element. If sentences are split into multiple elements, they should be combined. During processing, individual sentences will be extracted from text blocks to form units of analysis.
